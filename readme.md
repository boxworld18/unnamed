#### 第六章　學術不端、違反學習紀律的行為與處分

第二十一條 有下列違反課程學習紀律情形之一的，給予警告以上、留校察看以下處分：

- 課程作業抄襲嚴重的；
- 實驗報告抄襲嚴重或者篡改實驗數據的；
- 期中、期末課程論文抄襲嚴重的；
- 在課程學習過程中嚴重弄虛作假的其他情形。
